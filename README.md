## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/houses_page.git
```

Go to the project directory

```bash
  cd houses_page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
